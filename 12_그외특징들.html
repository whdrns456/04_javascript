* 호이스팅 (hoisting)

모든 변수 선언문이 코드의 최상단으로 이동되는 것처럼 느껴지는 현상 var 타입에서 확인되며, let과 const 타입에서는 호이스팅 현상이 일어나지 않는 것처럼 동작되게 해준다

* 스코프 (scope)

변수 사용 가능 범위 - var : functional scope. 변수가 선언된 영역 내에서만 사용 가능
- let, const : block scope. 변수가 선언된 블럭({}) 내에서만 사용 가능 (자바랑 유사하다!)

* 전역변수, 지역변수

- 전역 변수 : 특정 함수영역 밖에 선언된 변수(어디서든 사용 가능!) - 지역 변수 : 특정 범위({}) 내에 선언된 변수

* 배열 관련 메소드

배열.map()

배열 내의 모든 요소 각각을 함수 결과 값으로 새로운 배열을 반환

배열.filter()

배열 내의 요소들을 주어진 함수에 의해 조건을 만족하는 항목만 모아 새로운 배열을 반환

배열.reduce()

배열 내의 요소들을 주어진 함수를 실행하고 하나의 결과값을 반환
