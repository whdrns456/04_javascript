자바스크립트 개요

1. 스크립트 언어란 ?

프로그램의 동작을 사용자 요구에 맞게 수행하도록 해주는 용도
빠르게 배울 수 있고 짧은 소스코드로 상호작용 되도록 고안됨

2. 자바스크립트

웹 브라우저에서 많이 사용되는 인터프리터방식의 스크립트 언어 => 인터프리터 방식 : 코드를 한줄씩 읽어가며 바로 실행하는 방식(실시간으로 텍스트 분석해서 실행됨) -> 코드에 문법 오류가 있으면 실행 시점에 알 수 있음

3. 장단점

* 장점

  • 컴파일 과정 없이 인터프리터로 코드를 한줄씩 읽어가면서 실행되기 때문에 수행속도가 빠르다
  • 코드 작성이 간단해서 처음 접할 때 수월하다. (단순한 구조, 원칙/규칙) ex) 접근제한자 개념이 없다, 변수 선언 시 자료형을 정해줄 필요가 없다

* 단점

  • 웹에 특화된 기술 -> 내부에서 제공하는 기능이 제한적일 수 있다
  • html 소스코드 내에 작성하면 외부에 공개된다 -> 보안취약점이 발생된다

4. 자바스크립트 사용법

  1. inline(인라인) : 태그 내에 간단한 소스코드를 작성해서 실행하는 방법
  2. internal(내부) : html 문서 내에 소스코드를 작성해서 실행하는 방법(script 태그)
  3. external(외부) : 별도의 js파일로 소스코드를 작성해서 현재 문서에 연결하여 실행하는 방법이다

1) 인라인(inline)

[표현법] <태그명 on이벤트명="해당 요소의 이벤트 발생 시 실행할 코드">



2) 내부 (internal)

script 태그 영역 안에 소스코드를 작성
-> head 태그, body 태그 내에 모두 작성 가능!



3) 외부 (external)

외부 파일(.js)을 작성하여 현재 문서에 연결
script 태그를 이용하여 연결!
